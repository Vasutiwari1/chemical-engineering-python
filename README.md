## 📊 Project 7: Chemical Plant KPI Dashboard
**File:** `chemical_plant_dashboard_data.xlsx`

Interactive Excel dashboard analyzing chemical reactor 
performance across 120 data points.

**Features:**
- 4 Pivot Tables (Reactor, Month, Operator, Batch)
- 3 PivotCharts (Bar, Line, Combo)
- Slicers for interactive filtering
- KPI Summary (Best Reactor, Avg Yield, Best Operator)

**Tools:** Microsoft Excel, Pivot Tables, PivotCharts
